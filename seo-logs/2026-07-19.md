## ⚠️ Incident — script automatisé indisponible

`node scripts/seo-daily.mjs` a échoué dès l'étape de génération de l'article :

```

Please go to Plans & Billing to upgrade or purchase credits."
```

**La clé `ANTHROPIC_API_KEY` fournie n'a plus de crédit.** Tant qu'elle n'est
pas rechargée (ou remplacée), toute exécution automatique du script
(notamment via GitHub Actions, si configuré) échouera de la même façon, à
la fois pour l'article du jour et pour l'optimisation des fiches voitures
les mardis/jeudis. À corriger en priorité côté Anthropic Console.

À noter également : la lecture Sanity via `@sanity/client` dans le script
échoue avec `tunneling socket could not be established, statusCode=403`
(proxy sortant bloqué dans cet environnement) — c'est pour cette raison que
le script est conçu pour retomber sur le connecteur MCP Sanity, qui lui
fonctionne normalement (utilisé ci-dessous).
